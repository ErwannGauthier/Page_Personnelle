2021 - Agend'App

Bot Discord : Python - API Discord.py - SQLite

Ce projet personnel est un Bot Discord permettant d'avoir un agenda sur un serveur, d'y ajouter des groupes/classes et des matières à l'aide de commandes simples.

Les admins d'un serveur ont la possibilité de créer des groupes et des matières.
Les membres d'un serveur peuvent rejoindre un ou plusieurs groupes.
Ils peuvent ajouter des devoirs pour un groupe (seuls ceux auxquels ils appartiennent), une matière et une date précise.
Ils peuvent consulter les devoirs des groupes auxquels ils appartiennent.

2021 - Application de gestion de stock d'un commerce

Application : Java & Java FX - SQLite

Projet de groupe réalisé dans le cadre de notre fin de première année de DUT.

L’application simule la gestion de stock d'un commerce. Elle comporte une partie admin et une partie client/visiteur.

L'admin a la possibilité de consulter, ajouter, modifier ou supprimer les produits et les clients présents dans la base de données.
Les clients ont la possibilité de consulter les produits en stock, de les ajouter à leur panier et d'acheter leur panier.
